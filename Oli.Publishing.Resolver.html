    Resolves a revision from a list of resource ids and a given context slug.

    Ensures that the ordering of the resolved revisions matches the ordering of the input resource ids. If any of the resource ids cannot be resolved, a nil will be present in the slot in the resultant list. In other words, this function can be viewed as a functional mapping of the list of resource ids to their resolved revision.
