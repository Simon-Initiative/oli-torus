    Functions

    authorization_link(conn, provider, opts \\ [])

    Generates an authorization link for a provider. The link is used to sign up or register a user using a provider. If :invited_user is assigned to the conn, the invitation token will be passed on through the URL query params.

    current_pow_config(conn)
    deauthorization_link(conn, provider, opts \\ [])

    Generates a provider deauthorization link. The link is used to remove authorization with the provider.

    get_pow_config(atom)
    provider_class(provider)
    provider_icon(provider)
    provider_links(conn, link_opts \\ [])

    Generates list of authorization links for all configured providers. The list of providers will be fetched from the PowAssent configuration, and authorization_link/2 will be called on each. If a user is assigned to the conn, the authorized providers for a user will be looked up with PowAssent.Plug.providers_for_current_user/1. deauthorization_link/2 will be used for any already authorized providers.
