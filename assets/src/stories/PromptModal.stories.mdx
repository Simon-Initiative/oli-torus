**_Note:_** This doc page specifies className="absolute" to prevent the modal from covering other content of the page and overlapping with each other,
this is not normally required as it's positioned fixed by default.

## Quick-use hook

There is a hook that makes the most common use-case of displaying a prompt modal easier.

```tsx
import { usePromptModal } from '../../components/misc/PromptModal';

const YourComponent = () => {
  const onReset = () => {
    // Do something
  };

  const { Modal, showModal } = usePromptModal(
    'Are you sure you want to reset the schedule to the default values?',
    onReset,
  );

  return (
    <div>
      Your component content.
      <button onClick={showModal}>Show Modal</button>
      {Modal}
    </div>
  );
};
```
