    Oli.Delivery.Attempts.PageLifecycle.AttemptState (OLI Torus v0.17.0) View Source

    The complete state of a page attempt

    resource_attempt - The resource attempt record itself

    attempt_hierarchy - The activity attempt and part attempt hierarchy in the form of a map of activity resource id to tuples {%ActivityAttempt, part_attempt_map}, where part attempt map is a map of part ids to part attempt records.

    A full example of the attempt_hierarchy for two activities each with two parts would look like:

    %{
  45 => {%ActivityAttempt{}, %{"1" => %PartAttempt{}, "2" => PartAttempt{}}},
  67 => {%ActivityAttempt{}, %{"1" => %PartAttempt{}, "2" => PartAttempt{}}}
}
