# Output Requirements

When completing a `spec_scenario` task, report:

1. Scenario coverage summary
- Behavior validated
- Scenario files added/updated
- ExUnit files added/updated

2. Guardrail compliance
- Explicit statement that no fixtures/factories/mocks were introduced for domain setup
- Execution path used (`Oli.Scenarios.execute*` API)

3. Verification
- Commands run
- Pass/fail outcomes
- Any remaining known gaps or risks

4. File references
- List changed files with clickable paths
