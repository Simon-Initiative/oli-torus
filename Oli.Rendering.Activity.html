    Oli.Rendering.Activity behaviour (OLI Torus v0.5.1) View Source

    This modules defines the rendering functionality for an Oli activity. Rendering is extensibile to any format which implements the behavior defined in this module, then specifying that format at render time. For an example of how exactly to extend this, see activity/html.ex.

    Link to this section Summary

    Functions

    render(context, element, writer)

    Renders an Oli activity given an activity-reference element and using the activity_map from context. Returns an IO list of raw html strings to be futher processed by Phoenix/BEAM writev.
