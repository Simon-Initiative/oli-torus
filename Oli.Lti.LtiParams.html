    create_or_update_lti_params(params, user_id \\ nil)

    View Source

    Creates or updates a users lti params for the given issuer, client_id, deployment_id, context_id and sub.

    If a user_id is given then the lti params will be created/update with the given id. If the user_id is omitted or nil, then the user id will remain the same (or nil if lti params are being created). Once created, only the params (JSON lti params), exp (expiration) and associated user_id can be updated. A user_id can not be set back to nil once it has been set.
