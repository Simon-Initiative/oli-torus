     Code Formatting

    This project uses the default Elixir code formatting rules through running mix format. To enable auto-format on save, install the ElixirLS plugin for your code editor and add a configuration option. For Visual Studio Code, you can do this by opening the config file with cmd+p, typing > Open Settings, and adding this line:

    • "editor.formatOnSave": true
