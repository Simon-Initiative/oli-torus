    explain(query, opts \\ [])

    View Source

    Explains the query plan for a given Ecto based query. Options to either inline log the result and return the query, or to just return to analyzed result. Results can be in either json, text, or yaml format. Default is to log output as JSON.

    Use this inline for development on a query by query basis like:

    from(s in Section,
      join: spp in SectionsProjectsPublications,
      on: s.id == spp.section_id,
      join: pr in PublishedResource,
      on: pr.publication_id == spp.publication_id,
      join: rev in Revision,
      on: rev.id == pr.revision_id,
      where:
        (rev.resource_type_id == ^page_id or rev.resource_type_id == ^container_id) and
          s.slug == ^section_slug,
      select: rev
    )
    |> Oli.Utils.Database.explain()
    |> Repo.all()
